# Project Overview
We desing this site to help students manage their schedule by giving them one site to check their upcoming deadlines and events for their courses. We focussed on keeping the site clear and usable while giving the user a useful tool to help them with their day to day life in school.

# Pages
1. DASHBOARD
This pages gives the user an overview of their schedule in the form of a calendar. certain events and deadlines will be added to the days that they happen on givng the user a clear view of the upcoming month and when everything takes place to help them budget their time accordingly. There is also 2 boxes on the side that give more detail on each event and deadline on the calendar which students can use to see exactly what event or task is taking place.

2. TASKS
This page lets users manage their task by either adding new tasks, deleting tasks that may have occured or are no longer happening and edit tasks that may have changed dates or time. Users can also filter the tasks by date to see the order in which the tasks will be done

3. COURSES
This page outlines which courses the usesr is enroilled in and allows them to add new courses. It also gives the user the date of the next deadline for the course and how many assignments they have for it.

4. EVENTS
This page gives users a chance to view each upcoming event they have registered in. This includes information like when and where they take place alongside a brief description of the event itself.

5. SETTINGS
This page gives users the ability to mange their preferences on the sife like how the want their notifications delivered or being able to change their password and manage how the site uses their data.

6. LOGIN/ACCOUNT
When clicking on account the user is firt brought to a page where they must sign in. Once signed in they can view their account information and which emails have been connected to the site.

7. NOTIFICATIONS
This page shows the user any notifiactions that may have been sent to them to remind them of upcoming events or deadlines they have.

8. SEARCH
This page allows users to search for new course, tasks or eventsto add to their account and calendar.
